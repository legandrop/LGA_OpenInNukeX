v1.73: Build macOS / AGL: arreglo de raiz del `ld: framework 'AGL' not found` y del mismatch de version que aparecia cuando el sistema exponia una stub construida para un macOS mas nuevo. El fallback `-framework AGL` lo agrega Qt6 via `WrapOpenGL::WrapOpenGL` (definido en `$QTDIR/lib/cmake/Qt6/FindWrapOpenGL.cmake`) por transitividad de `Qt6::Widgets`. En `QtClient/CMakeLists.txt` se agregan dos bloques: el preventivo `set(OPENGL_agl_LIBRARY "")` antes de `find_package(Qt6 ...)` y el purgador que quita AGL de `WrapOpenGL::WrapOpenGL` despues. En `QtClient/compilar.sh` y `QtClient/compilar_dev.sh` se elimina el dummy `AGL.framework` y el flag `-F $PWD`; `compilar_dev.sh` incluye migracion one-shot para limpiar el `-F` heredado en la cache. [Build macOS - Fix AGL sin dummy framework]
